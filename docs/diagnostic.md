# TBOX PROV Service 诊断规范

## 概述

本文档定义TBOX PROV服务的诊断接口规范，包括UDS服务、DID/RID定义、错误码映射等。

## UDS服务

### 诊断会话控制 (0x10)

支持以下会话类型：
- 默认会话 (0x01)
- 编程会话 (0x02)
- 扩展诊断会话 (0x03)

### 安全访问 (0x27/0x29)

#### 支持的安全级别

| 级别 | 说明 | 用途 |
|------|------|------|
| 0x27 | 标准安全访问 | VIN写入、配置写入 |
| 0x29 | 高级安全访问 | 授权重写 |

#### 安全访问流程

1. **请求种子**：Tester发送SecurityAccess请求（子功能=安全级别）
2. **返回种子**：ECU返回4字节种子
3. **计算密钥**：Tester根据种子计算密钥
4. **发送密钥**：Tester发送SecurityAccess请求（子功能=安全级别|0x80）+ 密钥
5. **验证结果**：ECU验证密钥并返回结果

#### 密钥计算算法

```
密钥 = f(种子, 安全级别, ECU UID)
```

具体算法在实现中定义。

### 读数据标识符 (0x22)

#### 支持的DID

| DID | 名称 | 数据长度 | 说明 |
|-----|------|----------|------|
| 0xF190 | VIN | 17字节 | 车辆识别码（ASCII） |
| 0xF191 | VEHICLE_BINDING | 可变 | 车辆绑定信息（JSON） |
| 0xF192 | PROVISION_STATE | 1字节 | 个性化状态 |
| 0xF193 | VEHICLE_CONFIG | 可变 | 车辆配置数据 |
| 0xF194 | PRODUCTION_INFO | 可变 | 生产信息（JSON） |

#### 响应格式

**成功响应：**
```
[SID + 0x40] [DID High] [DID Low] [Data...]
```

**否定响应：**
```
[0x7F] [SID] [NRC]
```

### 写数据标识符 (0x2E)

#### 支持的DID

| DID | 名称 | 数据长度 | 说明 |
|-----|------|----------|------|
| 0xF190 | VIN | 17字节 | 车辆识别码（ASCII） |
| 0xF193 | VEHICLE_CONFIG | 可变 | 车辆配置数据 |
| 0xF194 | PRODUCTION_INFO | 可变 | 生产信息（JSON） |

#### 请求格式

```
[0x2E] [DID High] [DID Low] [Data...]
```

#### 响应格式

**成功响应：**
```
[0x6E] [DID High] [DID Low]
```

**否定响应：**
```
[0x7F] [0x2E] [NRC]
```

### 例程控制 (0x31)

#### 支持的RID

| RID | 名称 | 说明 |
|-----|------|------|
| 0xFF00 | REWRITE_ROUTINE | 授权重写例程 |

#### 请求格式

```
[0x31] [SubFunction] [RID High] [RID Low] [Data...]
```

#### 响应格式

**成功响应：**
```
[0x71] [SubFunction] [RID High] [RID Low] [Data...]
```

**否定响应：**
```
[0x7F] [0x31] [NRC]
```

| NRC | 名称 | 说明 |
|-----|------|------|
| 0x11 | 服务不支持 | 请求的UDS服务不支持 |
| 0x12 | 子功能不支持 | 请求的子功能不支持 |
| 0x13 | 请求长度错误 | 请求消息长度错误 |
| 0x14 | 响应长度错误 | 响应消息长度错误 |
| 0x22 | 条件不正确 | 执行条件不满足 |
| 0x24 | 请求顺序错误 | 请求顺序错误 |
| 0x25 | 请求超时 | 请求处理超时 |
| 0x31 | 请求超出范围 | 请求参数超出范围 |
| 0x33 | 安全访问被拒绝 | 安全访问未通过 |
| 0x35 | 无效密钥 | 安全访问密钥无效 |
| 0x36 | 超过尝试次数 | 安全访问尝试次数超限 |
| 0x37 | 时间延迟未过 | 安全访问时间延迟未过 |
| 0x70 | 上传下载不支持 | 上传/下载操作不支持 |
| 0x71 | 传输数据暂停 | 数据传输暂停 |
| 0x72 | 一般编程失败 | 编程操作失败 |
| 0x73 | 响应挂起 | 响应挂起 |
| 0x7E | 当前会话子功能不支持 | 当前会话下子功能不支持 |
| 0x7F | 当前会话服务不支持 | 当前会话下服务不支持 |

## 诊断流程

### VIN写入流程

1. **进入扩展会话**
   - Tester: 10 03
   - ECU: 50 03

2. **安全访问 (0x27)**
   - Tester: 27 01 (请求种子)
   - ECU: 67 01 [Seed]
   - Tester: 27 02 [Key] (发送密钥)
   - ECU: 67 02 (成功)

3. **写入VIN**
   - Tester: 2E F1 90 [VIN数据]
   - ECU: 6E F1 90 (成功)

4. **读取VIN验证**
   - Tester: 22 F1 90
   - ECU: 62 F1 90 [VIN数据]

5. **读取绑定信息**
   - Tester: 22 F1 91
   - ECU: 62 F1 91 [绑定信息]

### 授权重写流程

1. **进入扩展会话**
   - Tester: 10 03
   - ECU: 50 03

2. **安全访问 (0x29)**
   - Tester: 29 01 (请求种子)
   - ECU: 69 01 [Seed]
   - Tester: 29 02 [Key] (发送密钥)
   - ECU: 69 02 (成功)

3. **执行重写例程**
   - Tester: 31 01 FF 00 [新VIN数据]
   - ECU: 71 01 FF 00 (成功)

4. **读取VIN验证**
   - Tester: 22 F1 90
   - ECU: 62 F1 90 [新VIN数据]

### 配置写入流程

1. **进入扩展会话**
   - Tester: 10 03
   - ECU: 50 03

2. **安全访问 (0x27)**
   - Tester: 27 01 (请求种子)
   - ECU: 67 01 [Seed]
   - Tester: 27 02 [Key] (发送密钥)
   - ECU: 67 02 (成功)

3. **写入配置**
   - Tester: 2E F1 93 [配置数据]
   - ECU: 6E F1 93 (成功)

4. **读取配置验证**
   - Tester: 22 F1 93
   - ECU: 62 F1 93 [配置数据]

## 错误处理

### 常见错误场景

1. **安全访问失败**
   - 原因：密钥错误、尝试次数超限
   - 处理：重置安全访问状态，重新尝试

2. **VIN格式错误**
   - 原因：长度错误、字符非法、校验位错误
   - 处理：返回NRC 0x31，提示VIN格式错误

3. **写保护冲突**
   - 原因：已锁定状态下尝试写入
   - 处理：返回NRC 0x33，需要授权重写

4. **存储错误**
   - 原因：NVM写入失败、读取失败
   - 处理：返回NRC 0x72，重试或报错

### 错误恢复

1. **重试机制**
   - 最大重试次数：3次
   - 重试间隔：100ms

2. **状态恢复**
   - 异常状态下自动恢复到安全状态
   - 记录错误日志用于故障诊断
